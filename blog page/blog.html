Important FAQ

What is the document object model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content

How do you select an element from the DOM?

To select an element from the DOM (Document Object Model) in JavaScript, you can use several methods provided by the document object, depending on how you want to identify the element. Here are some common ways to select elements:


  • By ID: Use getElementById() to select an element by its ID.
  • By Class Name: Use getElementsByClassName() to select elements by their class name. This returns a live HTMLCollection (an array-like object).
  • By Tag Name: Use getElementsByTagName() to select elements by their tag name. This also returns a live HTMLCollection.
  • By Query Selector: Use querySelector() to select the first element that matches a CSS selector.
  • By Query Selector All: Use querySelectorAll() to select all elements that match a CSS selector. This returns a static NodeList.
What is the event delegation in the context of the DOM, and why is it useful?

Event Delegation in the DOM refers to the technique of attaching an event listener to a parent element instead of directly to each child element. This allows you to handle events for multiple elements efficiently, especially when the number of child elements is dynamic or large.


Why is it useful?

  • Improved Performance: When dealing with a large number of elements, attaching event listeners directly to each element can lead to performance issues. Event delegation reduces the number of event listeners, improving performance.
  • Dynamic Elements: If elements are added or removed from the DOM dynamically, event delegation ensures that new elements automatically inherit the event listener from their parent.
  • Code Simplification: Event delegation can simplify your code by centralizing event handling logic in a single place.
How do you manipulate an element's attribute, styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute or by directly accessing properties on the element, such as element.src for images. To change an element style, you can modify its style property, allowing you to set CSS styles directly in JavaScript.
